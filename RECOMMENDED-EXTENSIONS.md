# Extensions recommandées pour PharmAlert

- **One Dark Pro** — Thème sombre élégant (déjà configuré dans .vscode/settings.json)

- **Flutter** — Framework
- **Awesome Flutter Snippets** — Raccourcis de code Flutter
- **Dart Data Class Generator** — Génère des data classes

### Python (Backend)
- **Python** — Official
- **Pylance** — Serveur de langage Python
- **Black Formatter** — Formatage automatique
- **Python Debugger** — Debug

### Mobile
- **Flutter Intl** — Internationalisation Flutter
- **Riverpod Snippets** — State management

### UI/UX
- **Error Lens** — Affiche les erreurs inline
- **Error Reports** — Rapports d'erreurs améliorés

- **ESLint** — Linting JS/TS

## ⚡ Productivité

- **Thunder Client** — Client API (alternative Postman)
- **Color Highlight** — Montre les couleurs dans le code
- **Material Theme Icons** — Icônes Material Design

## 📱 Android/iOS
- **Xcode Keymap** — Raccourcis Xcode dans WindSurf
- **Apple Developer Toolkit** — Outils Apple

---

## Installation rapide

Ouvre WindSurf → `Cmd + Shift + P` → tape `Extensions: Show Recommended Extensions`

Ou installe directement :
```
Cmd + Shift + P → "Extensions" → installe chaque extension par son nom ci-dessus
```

## Thème des couleurs activé
**One Dark Pro** — Thème Darwin/One Dark
